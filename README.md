# Portfolio Website

This is my personal portfolio website, built using **React** and **TypeScript** with **Tailwind CSS** for styling. It showcases about myself, experience, skills, projects, education, and contact achievements in a modern, responsive design.

## Features

- **React**: The app is built with React to create a dynamic and interactive user interface.
- **TypeScript**: TypeScript is used for better development experience with type-checking and ensuring code quality.
- **Tailwind CSS**: Tailwind CSS is used for styling and building beautiful UI components with a responsive design out-of-the-box.
- **Responsive Design**: The portfolio is fully responsive and works seamlessly across different devices, including mobile, tablet, and desktop.
- **Sending message**: Anyone can send message filling up a form.
- **Smooth Navigation**: Easy navigation to different sections like About, Experience, Skills, Projects, Education, and Contact.

## Technologies Used

- React
- TypeScript
- Tailwind CSS
